# 目录总览

本目录以“学科领域 + 主题 + 资源类型”组织资源，方便按学习阶段与使用场景检索。

## 资源类型

- 教材与讲义
- 法条与司法解释
- 案例库与判例摘要
- 论文与专著
- 课程与公开课
- 实务材料与指南

## 目录结构

- 法学基础
- 民商法
- 刑法
- 行政法与公共法
- 诉讼与程序法
- 国际法
- 经济法与金融法
- 比较法与科技法

## 三大学习模块入口

## 使用建议

1. 先确定你所处的学习阶段。
2. 在对应领域中按资源类型筛选。
3. 记录来源与版本，方便后续复盘。
