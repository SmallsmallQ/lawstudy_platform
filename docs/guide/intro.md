# 平台介绍

本平台用于汇集和维护法律学习资料的结构化目录，重点关注“定位效率”“复盘效率”和“引用可靠性”。内容本身由社区共建，平台提供组织方式、检索方式与标准化模板。

## 适用人群

- 法学本科生、研究生
- 法考备考者
- 法律从业者与研究者
- 对法律交叉学科感兴趣的学习者

## 你能在这里获得什么

- 结构化资源目录
- 可靠的来源记录与引用建议
- 学习路径与阶段目标
- 可复用的笔记与写作模板

## 内容标准

- 明确来源与版本
- 记录适用场景与难度
- 以可复用为目标撰写说明
- 尊重版权和引用规范

## 快速开始

1. 进入 `资源目录总览` 找到你的学科或目标主题。
2. 打开对应条目查看资源类型与推荐顺序。
3. 使用 `写作与笔记模板` 形成可复盘的学习记录。
4. 发现缺口后，按 `提交流程` 完善目录或补充资源。
