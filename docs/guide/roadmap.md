# 学习路径

不同阶段的目标不同，建议先明确角色，再根据路径选取资源。

## 本科基础路径

- 目标：建立学科框架，掌握基础概念与核心制度
- 推荐顺序：法理学 → 宪法 → 民法 → 刑法 → 行政法 → 诉讼法
- 输出形式：课程笔记 + 重点制度卡片

## 法考路径

- 目标：知识点覆盖与解题效率
- 推荐顺序：以考试科目为主线，配合真题与精讲
- 输出形式：错题本 + 高频法条清单

## 研究生路径

- 目标：形成研究问题、掌握研究方法
- 推荐顺序：基础理论 → 研究法课程 → 主题文献精读
- 输出形式：研究问题清单 + 文献综述模板

## 实务路径

- 目标：快速定位法条、案例与审判规则
- 推荐顺序：领域知识 → 典型案例 → 办案规则
- 输出形式：案例要旨卡片 + 办案流程清单

## 交叉学科路径

- 目标：建立法律与技术、经济、政策之间的结构化理解
- 推荐顺序：基础法学 → 目标领域概览 → 交叉问题文献
- 输出形式：问题树 + 术语表
