## 快速开始

1. 从 `资源目录总览` 进入，找到你当前的学科或目标主题。
2. 根据 `学习路径` 确定阶段目标，再挑选对应资源。
3. 用 `检索与标注` 规范你的文件命名与标签，便于长期积累。
4. 如果有新增资料，按 `提交流程` 提交并补充元数据。

## 平台定位

本项目聚焦“资源结构化与可检索”，而非重复制造内容。我们希望你在这里找到：

- 体系化目录
- 可靠的引用方式
- 可复用的学习流程

## 贡献指引

从修正目录名称到补充课程资料，任何贡献都欢迎。请先阅读 `贡献指南` 和 `提交流程`。
