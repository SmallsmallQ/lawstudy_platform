## 前言：构建开放的法律知识库

法律学习资源共享平台是一个面向法学学人、法律从业者及自学者的开源数字化知识库。本项目由西安交通大学法学院学生发起，旨在解决法学学习与研究过程中资源碎片化、检索效率低下的问题。平台通过系统化的目录结构，整合从法学基础理论到前沿交叉学科的研究资料，构建一个开放、持续更新的学术共同体。

## 为什么建立这个平台？

传统的法学资料往往散落于不同的数据库、学术网站或个人笔记中，缺乏结构化的整理，导致学习和研究的时间成本大幅增加。同时，随着技术的发展，法学研究的工具生态正在发生剧烈演变。本平台将分散的专业知识、案例数据以及现代研究工具链进行重新组织，为使用者提供系统性的信息支持。

## 本指南包含哪些内容？

为满足不同群体的学习与研究目标，本指南目前包括以下三大核心模块：

- **法学生必备技能库（工具箱）**：排版与写作生态（LaTeX、Markdown、引用管理）、AI 辅助研究（翻译、摘要、检索增强）、法律检索与数据获取（数据库高阶检索与清洗）、个人学术基础设施（文献管理、期刊订阅、环境搭建）。[进入技能库](/skills/overview)  
- **基础法学门径**：系统梳理民商法、刑法、行政法等核心部门法，配套关键法条解析、司法解释与典型案例的裁判逻辑拆解。[进入基础法学](/fundamentals/overview)  
- **前沿交叉学科探索**：聚焦人工智能法体系与计算法学，涵盖技术规制、全球治理动态、法学实证研究方法与法律数据应用逻辑。[进入前沿交叉](/frontier/overview)

## 参与贡献

本平台的建设依赖于学术共同体的协作。如果你在阅读过程中发现任何错漏，或者希望补充最新的学术资源与工具教程，欢迎通过 GitHub 提交 Issue 或 Pull Request 加入贡献者行列。
