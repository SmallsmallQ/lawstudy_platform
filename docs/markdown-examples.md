# 资源目录示例

本页示例展示条目如何填写，以便统一规范与检索。

## 教材条目示例

- 资源名称：民法学（某版本）
- 资源类型：教材
- 来源与版本：出版社与年份
- 推荐理由：框架清晰、适合作为入门
- 适用场景：本科基础课程
- 标签：民法、基础、教材

## 案例条目示例

- 资源名称：合同纠纷典型案例
- 资源类型：案例
- 来源与版本：裁判文书或案例汇编
- 推荐理由：涵盖关键争点与规则适用
- 适用场景：民商法进阶
- 标签：民法、合同、案例

## 论文条目示例

- 资源名称：某主题研究论文
- 资源类型：论文
- 来源与版本：期刊与年份
- 推荐理由：提出明确研究问题与方法
- 适用场景：研究生与专题学习
- 标签：法理学、方法论、研究
