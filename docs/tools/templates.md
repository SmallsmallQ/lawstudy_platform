# 写作与笔记模板

以下模板用于形成可复盘的学习记录。建议直接复制使用。

## 读书笔记模板

```md
- 书名：
- 作者：
- 版本/年份：
- 主题与问题意识：
- 核心观点：
- 关键概念：
- 结构摘要：
- 与其他文献的差异：
- 可引用结论：
```

## 案例分析模板

```md
- 案由与事实：
- 争点：
- 裁判规则：
- 法条依据：
- 评析与启示：
- 关键词：
```

## 文献综述模板

```md
- 研究主题：
- 代表性文献：
- 研究方法：
- 主要观点分歧：
- 研究空白：
- 可继续研究的问题：
```

## 课程笔记模板

```md
- 课程：
- 课次：
- 核心概念：
- 关键制度：
- 课堂案例：
- 待补充问题：
```
